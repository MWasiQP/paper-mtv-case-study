# Case Study: LLM-based multi-agent worfkflow concept for TSAD analysis using MTV

This repository contains materials for the case study mentioned in the corresponding paper that could not be included or discussed in detail in the main manuscript due to space limitations. 

## Relation to the paper
The material in this repository accompanies the paper and extends the brief case-study description reported in the Results section. It follows the case-study-style methodology and visual analytics tool, Visual Analytics for Detecting, Investigating, and Annotating Anomalies in Multivariate Time Series (MTV), introduced in the [1] and illustrates the use of multimodal large language models (MLLMs) as role-specialized users to support a multi-stage anomaly investigation process using the MTV workflow. 

## Contents
- 'case-study/': detailed description of the case study.
- 'assets/': screenshots from the MTV dashboard used in the case study workflow.

## Scope
This case study is presented as a demonstration of using a multi-agent, LLM-based anomaly investigation workflow with a time-series visualization and analysis tool. It includes dashboard views, and outputs related to the MTV-based multi-agent workflow with MLLMs.
